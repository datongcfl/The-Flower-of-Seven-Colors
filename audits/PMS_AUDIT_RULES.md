## 三、安全规则
### 3.1 OWASP API Security Top 10 (2023)
- [ ] API1 对象级授权失效（BOLA / IDOR）
- [ ] API2 鉴权失效（旁路认证、令牌缺陷）
- [ ] API3 对象属性级授权（批量分配 / 越权改字段）
- [ ] API4 资源消耗无限制（未限流、DoS）
- [ ] API5 功能级授权失效（普通用户调管理接口）
- [ ] API6 敏感业务流暴露（如无限次试订/占房）
- [ ] API7 SSRF（服务端请求伪造）
- [ ] API8 安全配置错误（默认口令、暴露调试端）
- [ ] API9 资产管理不当（影子 API / 废弃接口）
- [ ] API10 日志与监控不足（无审计日志）

### 3.2 合规基线
- [ ] PCI-DSS：支付卡数据不落库明文、传输加密
- [ ] 等保 2.0：身份鉴别、访问控制、安全审计（三级要求）
